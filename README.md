# Tài chính Gia đình (Family Finance App)

Ứng dụng quản lý tài chính cá nhân/gia đình được thiết kế theo phong cách UI phẳng, hiện đại (Stitch-inspired), sử dụng HTML, CSS và JavaScript thuần túy.

## Tính năng
- **Tổng quan Dashboard:** Hiển thị tổng số dư, thu nhập, chi tiêu.
- **Báo cáo Thu/Chi:** Biểu đồ thể hiện chi tiêu mô phỏng trong 7 ngày gần nhất.
- **Phân bổ ngân sách:** Theo dõi các khoản chi theo nhóm mục.
- **Quản lý giao dịch:** Thêm mới và xem danh sách các khoản thu chi, lưu trữ trực tiếp trên trình duyệt (localStorage).

## Cài đặt và Sử dụng
Dự án không yêu cầu cài đặt Node.js hay framework phức tạp.
1. Clone dự án về máy: `git clone https://github.com/banhdaidung-ai/Taichinh.git`
2. Mở file `index.html` trên bất kỳ trình duyệt web nào (Chrome, Safari, Edge...).
3. Sử dụng ngay lập tức! Mọi dữ liệu sẽ được lưu tự động trên trình duyệt của bạn thông qua `localStorage`.

## Công nghệ sử dụng
- **HTML5 & Vanilla CSS** cho cấu trúc và giao diện hiện đại (CSS Variables, Flexbox, Grid).
- **Vanilla JavaScript** cho toàn bộ logic trạng thái ứng dụng.
- **Chart.js** cho biểu đồ báo cáo.
- **Phosphor Icons** cho hệ thống biểu tượng.
